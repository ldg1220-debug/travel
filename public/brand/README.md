## 로고 원본에 대한 메모
현재 사이드바/스플래시/로그인은 `tradule-logo.png` 를 우선 사용하고, 없으면
✈️ 그라디언트 마크 + "Tradule 트레쥴" 워드마크로 폴백합니다.

업로드해주신 그래피티 로고는 **검은색 라인아트 + 넓은 여백**이라, 그대로 쓰면
작은 헤더에서 디테일이 안 보이고 흰 배경에서만 자연스럽습니다. 권장:
1. 로고 주변 **여백을 잘라내고**(tight crop),
2. 배경을 **투명 PNG**로 만들고,
3. 어두운 화면에서도 쓰려면 **흰색 버전**도 하나 준비.

파일을 이 폴더에 커밋하면 끝입니다 — 코드 수정 불필요.
